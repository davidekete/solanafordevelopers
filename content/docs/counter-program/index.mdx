## Module Goals

By building this project you will learn:

- How to <LinkPrev href={"/docs/counter-program/program-state"}>represent and manage onchain state using Rust structs</LinkPrev>.
- How to <LinkPrev href={"/docs/counter-program/program-instructions"}>create instructions and execute them with transactions</LinkPrev>.
- How accounts are created, how to allocate and calculate space the account occupies and how to make it rent exempt.
- How to deploy Solana programs on Localnet.
