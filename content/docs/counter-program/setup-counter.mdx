Here, you will scaffold a new Rust project for your counter program.

<Steps>
<Step>
### Create new Rust project
To create a Solana program, you have to start by initializing a new Rust project by running the command below:

```bash title="shell"
cargo init counter_program --lib
```

</Step>

<Step>
### Install Solana program dependencies
`cd` into the created project directory and run the command below to install the required crates:

```bash title="shell"
cargo add solana-program@2.3.0 solana-system-interface@1.0.0 borsh
```

</Step>
</Steps>

The dependencies you installed with the command above include:

- <LinkPrev href={"https://crates.io/crates/solana-program"}>`solana-program`</LinkPrev>: This crate includes the core runtime API for writing Solana programs. You need it to access Solana-specific types, macros, and functions required to write and run programs on-chain.
- <LinkPrev href={"https://crates.io/crates/solana-system-interface"}>`solana-system-interface`</LinkPrev>: This crate contains instructions and constructors for interacting with the <LinkPrev href={"https://docs.solanalabs.com/runtime/programs#system-program"}>System program</LinkPrev>. You need it to create new accounts, allocate account data, assign accounts to owning programs, etc.
- <LinkPrev href={"https://crates.io/crates/borsh"}>`borsh`</LinkPrev>: This crate provides a fast and compact binary serialization format. You need it to serialize and deserialize instruction data and account state for efficient storage and communication within your program.

Now that you have set up your program dependencies, you are ready to define your program’s state.
